# Admin Guide

## Responsibilities
- Approve deployment windows
- Manage update rollout percentages
- Coordinate license provisioning with sales
- Validate release readiness before customer onboarding

## Operational checklist
1. Confirm release checklist is complete
2. Confirm signing credentials are valid
3. Confirm update host contains the new artifacts
4. Enable staged rollout in increments
5. Monitor support and telemetry for 24 hours before expanding rollout
