# Licensing Operations

## Scope
This section documents commercial license creation, handling, and customer activation support.

## Included procedures
- Sales license request workflow
- Secure key generation handoff
- Customer onboarding activation sequence
- Lost device and rebinding support

## Security notes
- License keys must be generated only by authorized staff
- Generated keys must be transmitted through approved CRM or secure delivery channels
- Never distribute raw signing credentials or server secrets to sales personnel
