# Commercial Release Package

This folder contains non-code deliverables required for a commercial SPARELINK release.

## Included assets
- `RELEASE_CHECKLIST.md`
- `CHANGELOG_TEMPLATE.md`
- `SALES_LICENSE_KEY_WORKFLOW.md`
- `CUSTOMER_ONBOARDING_TEMPLATE.md`

## Binary artifacts
Desktop installers are emitted under `apps/desktop/release/<version>/` by electron-builder.
